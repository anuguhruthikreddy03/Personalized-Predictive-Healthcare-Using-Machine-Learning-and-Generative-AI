# 🏥 Personalized Predictive Healthcare Using Machine Learning & Generative AI

An intelligent healthcare prediction system that uses Machine Learning and Generative AI to provide personalized disease prediction, symptom analysis, and healthcare recommendations.

The system helps users analyze symptoms, predict possible diseases, and receive AI-powered healthcare guidance in an interactive and user-friendly way.

---

# 🚀 Features

- Disease prediction using Machine Learning
- Personalized healthcare recommendations
- Symptom-based disease analysis
- AI-powered healthcare assistance
- User-friendly web interface
- Multiple ML models integration
- Django-based backend
- Healthcare data processing

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/anuguhruthikreddy03/Personalized-Predictive-Healthcare-Using-Machine-Learning-and-Generative-AI.git
```

---

## Move to Project Folder

```bash
cd Personalized-Predictive-Healthcare-Using-Machine-Learning-and-Generative-AI
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv env
env\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv env
source env/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

```bash
python manage.py runserver
```

Open browser:

```text
http://127.0.0.1:8000/
```
